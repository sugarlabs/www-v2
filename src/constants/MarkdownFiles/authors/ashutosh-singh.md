# About Ashutosh Singh

Hi! I'm Ashutosh Singh, a software developer and open-source contributor from India. I'm currently a GSoC 2026 contributor at Sugar Labs, working on the Sugar Activity on Demand project, an AI-powered system that lets learners describe a learning activity in plain language and generates a real, installable Sugar Activity from it.

## Experience

- **GSoC 2026 Contributor at Sugar Labs**  
  Building Sugar Activity on Demand, a constructionist AI tool that generates GTK3-based Sugar Activities from natural language prompts, complete with templates, licensing, and a Reflective Studio for learner-driven code exploration.

## Current Projects

- **Sugar Activity on Demand**  
  An end-to-end system combining LLM-powered code generation with constructionist learning principles. Features a GTK3 frontend with a Prompt Screen, Reflective Studio (AI co-designer, code review, version history), and a backend generator that produces installable `.xo` bundles.
